HEIDIEditor
===========

HTML5 WYSIWYG Editor for NLM-XML  (Standalone Application or as OMP Plugin)

### Dependencies
  1. [meTypeset](https://github.com/MartinPaulEve/meTypeset)
  2. [xmltodict](https://github.com/martinblech/xmltodict)
